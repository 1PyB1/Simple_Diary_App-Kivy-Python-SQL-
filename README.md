# Simple Diary App (Kivy + Python + SQL)

A simple and clean Diary Application built using **Python**, **Kivy**, and **SQL**.  
The app allows users to store diary pages with dates, manage multiple entries, and navigate between them through a modern UI.  
A running MySQL Server is required. Make sure to create the necessary database and update your connection settings inside Database.py.

## Features

- Create and edit diary pages  
- Save date and text content  
- User authentication system (Sign In / Sign Up)  
- Clean and minimal Kivy interface  
- SQL database integration  
- Includes custom fonts and image resources

## Installation

1. Make sure **Python 3.x** is installed on your system.
2. Install required packages
```bash
pip install kivy mysql-connector-python
```
```bash
python Diary.py
```
